# Home Dashboard — Домашняя панель на e-ink дисплее

Модульная система домашнего дашборда. Сервер собирает данные из внешних API, рендерит дашборд как HTML/CSS-страницу, конвертирует её в 1-битное PNG-изображение и раздаёт по локальной сети. Микроконтроллер ESP32 с e-paper дисплеем каждые 10 минут скачивает и отображает это изображение.

Основные сценарии:
- Отображение лимита токенов на CloudCode, Codex
- Отображение продуктивности на wakatime.com
- Отображение календаря на текущий день

## Как это работает

```
API (AI, погода, календарь, авто)
        ↓
    сервер
  (сбор + кэш данных)
        ↓
  HTML/CSS → PNG (1-bit)
        ↓
       сеть
        ↓
  ESP32 + e-paper дисплей
```

1. Сервер опрашивает включённые сервисы по расписанию и кэширует результаты.
2. При запросе `/dashboard/image` сервер делает скриншот HTML-страницы и конвертирует его в 1-битный PNG.
3. ESP32 скачивает изображение по HTTP каждые 10 минут, ночью (00:00–05:00) уходит в сон для экономии батареи.
