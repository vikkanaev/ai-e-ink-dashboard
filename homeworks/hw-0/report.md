## 3–5 уточнений для следующего захода

1. Явно запретить коммиты без подтверждения в промпте: «не коммить без явной просьбы»
2. Не устанавливать глобальные инструменты без явного запроса — только диагностировать
3. При выборе вариантов давать не односложные ответы (C/B), а пояснять причину выбора —
   агент учитывает контекст в последующих шагах
4. Добавить в промпт ссылку на конкретный источник агентов курса, чтобы не искал самостоятельно
5. Разделить задачи: сначала только проектирование CLAUDE.md, потом отдельной сессией — окружение
